Өмнө нь 2018 онд энэхүү аргыг MSExcel ашиглан матрицын аргаар тооцох аргыг оруулсан байгаа. Энэ удаа тэр жишээг Python хэл дээр бичсэн кодоор баяжууллаа. Python хэлний кодыг бичиж тайлбарлахдаа Jupiter Notebook ашигласан болно.

## Грумбрехтын арга

Грумбрехтын арга нь нүүрс баяжуулах үйлдвэрийн материалын балансыг тооцоход ашиглагддаг математик арга юм. Энэ арга нь олон бүтээгдэхүүн бүхий үйлдвэрийн урсгалуудын хооронд байгаа харилцааг тодорхойлох боломжийг олгодог.

## Python код ашигласан шийдэл

```python
import numpy as np
import pandas as pd

# Үйлдвэрийн параметрүүд
gamma = [55, 60, 75]  # Баяжмалын гарц (%)
ash = [12, 15, 45]    # Үнслэг агуулга (%)

# Матриц байгуулах
A = np.array([
    [1, 1, 1],
    [gamma[0]/100, gamma[1]/100, gamma[2]/100],
    [ash[0]/100, ash[1]/100, ash[2]/100]
])

# Тэжээлийн векторэфект  
b = np.array([100, 65, 25])

# Системийг шийдэх
solution = np.linalg.solve(A, b)

print(f"1-р бүтээгдэхүүн: {solution[0]:.2f}%")
print(f"2-р бүтээгдэхүүн: {solution[1]:.2f}%") 
print(f"3-р бүтээгдэхүүн: {solution[2]:.2f}%")
```

## Дүгнэлт

Python програмчлалын хэл нь нүүрс баяжуулалтын балансын тооцоог хийхэд маш хүчирхэг хэрэгсэл болдог. NumPy, Pandas зэрэг сангууд нь матрицын үйлдлүүдийг хялбар болгож, тооцооллын үр дүнг хурдан гаргаж өгдөг.
