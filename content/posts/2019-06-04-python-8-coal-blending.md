## Нүүрсний оновчтой хольц

Нүүрс баяжуулах үйлдвэрийн (НБҮ) технологийн үзүүлэлт анхдагч нүүрсний чанараас ихээхэн хамаардаг. Хэд хэдэн уурхай, эсвэл мөргөцөг ашигладаг ихэнх уурхайд чанарын холилт (шихт) хийдэг.

НБҮ-ийн тэжээлийн чанарын холилтыг ухаалагаар хийх нь үйлдвэрийн үр ашгийг дээшлүүлэх, технологийн үзүүлэлтүүдийг оновчлох нэг арга зам юм.

### Математик загвар

Иймэрхүү оновчлолын бодлогыг бодоход математикийн шугаман программчлалын (ШП) аргыг хэрэглэдэг. Python-ы PuLP пакетыг ашиглан шийдлийг харуулъя.

### Бодлого 1 - Баяжмалын гарцыг дээшлүүлэх

НБҮ-т хэд хэдэн уурхайгаас (1, 2, 3) нүүрс боловсруулж байгаа:

**Өгөгдөл:**
- 1-р уурхай: γ₁=55%, A₁=12%
- 2-р уурхай: γ₂=60%, A₂=15%  
- 3-р уурхай: γ₃=75%, A₃=45%

**Шаардлага:**
- Нийт тэжээл: Q = 8000 т/өдөр
- Үнслэг: A ≤ 18%
- Уурхайн хязгаарууд:
  - 500 ≤ Q₁ ≤ 1000
  - 1000 ≤ Q₂ ≤ 5000
  - 1000 ≤ Q₃ ≤ 2000

### Python шийдэл

```python
from pulp import *

# Бодлого үүсгэх
prob = LpProblem("Coal_Blending_Max_Yield", LpMaximize)

# Хувьсагчид
Q1 = LpVariable("Q1", 500, 1000)
Q2 = LpVariable("Q2", 1000, 5000)
Q3 = LpVariable("Q3", 1000, 2000)

# Зорилго функц - Баяжмалын гарцыг дээшлүүлэх
prob += 0.55*Q1 + 0.60*Q2 + 0.75*Q3, "Total_Concentrate"

# Хязгаарлалтууд
prob += Q1 + Q2 + Q3 == 8000, "Total_Feed"
prob += (0.12*Q1 + 0.15*Q2 + 0.45*Q3)/(Q1 + Q2 + Q3) <= 0.18, "Ash_Limit"

# Бодох
prob.solve()

# Үр дүн
print(f"Status: {LpStatus[prob.status]}")
print(f"Q1 = {value(Q1):.0f} т/өдөр")
print(f"Q2 = {value(Q2):.0f} т/өдөр")
print(f"Q3 = {value(Q3):.0f} т/өдөр")
print(f"Баяжмал = {value(prob.objective):.0f} т/өдөр")

# Үнслэг шалгах
ash = (0.12*value(Q1) + 0.15*value(Q2) + 0.45*value(Q3))/8000
print(f"Тэжээлийн үнслэг = {ash*100:.2f}%")
```

### Бодлого 2 - Үнслэгийг бууруулах

Баяжмалын гарцыг γ = 5200 т/өдөр гэж тогтоож, үнслэгийг багасгахыг зорино.

```python
prob2 = LpProblem("Coal_Blending_Min_Ash", LpMinimize)

# Хувьсагчид
Q1 = LpVariable("Q1", 500, 1000)
Q2 = LpVariable("Q2", 1000, 5000)
Q3 = LpVariable("Q3", 1000, 2000)

# Зорилго функц - Үнслэгийг багасгах
prob2 += 0.12*Q1 + 0.15*Q2 + 0.45*Q3, "Total_Ash"

# Хязгаарлалтууд
prob2 += Q1 + Q2 + Q3 == 8000, "Total_Feed"
prob2 += 0.55*Q1 + 0.60*Q2 + 0.75*Q3 == 5200, "Concentrate_Yield"

# Бодох
prob2.solve()

# Үр дүн
print(f"\nБодлого 2:")
print(f"Q1 = {value(Q1):.0f} т/өдөр")
print(f"Q2 = {value(Q2):.0f} т/өдөр")
print(f"Q3 = {value(Q3):.0f} т/өдөр")
print(f"Үнслэг = {value(prob2.objective)/8000*100:.2f}%")
```

## Дүгнэлт
